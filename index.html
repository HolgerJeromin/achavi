achavi - Augmented OSM Change Viewer - alpha

Displays updates to OpenStreetMap based on minutely Augmented Diffs provided by Overpass API.

Usage:

  1. press "bbox", then click and drag to draw bounding box (optional but recommended)
  2. press "live" to load current diff and activate minutely refreshing
  3. press "clear" to remove all loaded changes
  • you can also drag & drop diffs onto map from Overpass API
  • zoom on changes with shift + drag zoom box
